Jetty

Jetty is a well-known lightweight http server, servlet container and much more. It particularly fits with a system having limited resources such as a NAS

Java on Synology ?

Yes ! You can develop java applications on you Synology NAS. The only prerequisite is to install the Java package.

How To install ?

Just download the SPK file and in your Diskstation, open Package Manager, click install, select the SPK you just downloaded, wait a few seconds and...that's it your web server is up and running!!
